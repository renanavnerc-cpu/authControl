# AccessHub

AccessHub é uma plataforma de gestão de usuários e controle de acesso, com autenticação, permissões (roles) e controle de acesso por rota e ação. O projeto simula um ambiente real de aplicações SaaS, com foco em arquitetura escalável, organização de código e boas práticas de desenvolvimento.

## 🚀 Funcionalidades

- Autenticação de usuários
- CRUD completo de usuários
- Sistema de permissões (roles)
- Controle de acesso por rota e ação
- Listagem com busca e paginação
- Dashboard com métricas básicas

---

## 🛠️ Tecnologias

- Next.js
- TypeScript
- React
- Tailwind CSS
- Node.js / Java (Spring Boot)
- React Testing Library
- Cypress

---

## 🧠 Boas práticas aplicadas

- Separação de responsabilidades
- Componentização e reutilização de código
- Tipagem forte com TypeScript
- Tratamento de erros
- Código limpo e organizado
- Testes automatizados
